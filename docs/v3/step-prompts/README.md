# ATELIER OS V3 STEP PROMPTS

This folder contains ready-to-paste prompts for each step in `docs/v3/execute_v3.md`.

Use these when you want Cursor to implement the plan one exact step at a time.

How to use:

1. pick the phase
2. open the matching file in this folder
3. copy the step prompt
4. paste it into Cursor
5. let Cursor finish only that step
6. review before moving to the next step

- `PHASE_00_STEP_PROMPTS.md`
- `PHASE_01_STEP_PROMPTS.md`

Golden rule:

- one prompt = one step
- do not combine steps unless the repo proves they are inseparable
- stop after each step
